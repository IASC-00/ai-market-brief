# Project 2: AI Market Brief

## Live Demo
**[ai-market-brief.onrender.com](https://ai-market-brief.onrender.com)**

## Service Demonstrated
**AI Research & Reporting Automation** — clients get on-demand AI-generated market
briefs, competitive summaries, and business intelligence reports without paying
for an analyst or spending hours on manual research.

## Goal
Build a demo tool that generates a structured, readable market brief on any topic
or industry in seconds — showing the kind of automated research output a real
client would receive on a weekly or on-demand basis.

## Stack
- Python / Flask
- OpenRouter API (free tier)
- Render (free tier deployment)

## Run Locally
```bash
pip install -r requirements.txt
OPENROUTER_API_KEY=your_key python app.py
```
